# homebrew-iflow-cli

Homebrew tap for installing [`iflow-cli`](https://github.com/iflow-ai/iflow-cli).

## Install

```bash
brew tap iflow-ai/iflow-cli
brew install iflow-cli
```

## Upgrade

```bash
brew update
brew upgrade iflow-cli
```

## Verify

```bash
iflow --version
```

## Formula

The formula is located at:

- `Formula/iflow-cli.rb`
